🚀 Panduan Setup

Ikuti langkah-langkah berikut untuk mengaktifkan sistem

📋 Langkah-Langkah Setup

1

Buka Google Sheets Baru

Pergi ke sheets.google.com dan buat spreadsheet baru.

2

Buka Apps Script Editor

Klik menu Extensions → Apps Script untuk membuka editor kode.

3

Paste Kode GS

Hapus kode default, kemudian paste seluruh isi file Code.gs yang sudah didownload. Klik tombol Save (💾).

4

Jalankan setupSheet()

Di dropdown fungsi, pilih setupSheet lalu klik ▶ Run. Izinkan akses yang diminta Google.

5

Deploy sebagai Web App

Klik Deploy → New deployment → pilih type Web app. Atur "Who has access" ke Anyone. Klik Deploy dan copy URL-nya.

6

Setup Auto Trigger

Jalankan fungsi setupTrigger agar email terkirim otomatis setiap hari jam 08:00 WIB.

7

Paste URL ke Dashboard

Buka file dashboard.html dengan text editor, cari baris const API_URL = "..." di bagian script, ganti nilai URL dengan URL deployment yang sudah dicopy. Simpan file, lalu buka di browser.

⚠️ Catatan Penting

  • 🔑 Gmail/Google Workspace diperlukan. Email dikirim menggunakan GmailApp, gratis hingga 100 email/hari.
  • 🔒 Dashboard berjalan 100% di browser, data tersimpan di Google Sheets Anda sendiri.
  • 🌐 CORS sudah ditangani dengan JSONP. Dashboard dapat dihosting di mana saja.
  • ♾️ Setelah deploy, sistem bekerja sepenuhnya gratis tanpa biaya server.
